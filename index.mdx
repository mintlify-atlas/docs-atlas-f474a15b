## Architecture at a Glance

The system is organized into four independent modules:

<Steps>
  <Step title="Core Module">
    Contains pure Java business logic, domain models, and port interfaces. Zero framework dependencies.
  </Step>
  <Step title="Data Module">
    Implements repository ports with multiple persistence adapters: JPA, JDBC, MongoDB, and Elasticsearch.
  </Step>
  <Step title="Web Module">
    Provides REST API adapters with Spring Web controllers and DTOs.
  </Step>
  <Step title="App-Main Module">
    Bootstrap module that wires everything together and launches the Spring Boot application.
  </Step>
</Steps>
